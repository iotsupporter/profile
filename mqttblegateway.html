BLE MQTT Gateway

This is the MQTT BLE Gateway. The product uses the ESP32 SoC Chip and BT840XE and it transfer the data of peripheral bluetooth device to cloud server. Also it sends the user command to bluetooth device. The product is manufactured by order in Zhuhai Factory.

What I did

  • PCB Design by Altium Designer
  • ESP32 Firmware using ESP-IDF
  • nRF52840 Firmware using Zephyr
  • Manufacture the Product
