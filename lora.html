LoRa Node and Gateway

The ESP32 is worked as LoRa node and Raspberry Pi is worked as Gateway. TTN is also used. The ESP32 can be connected to TTN and send the data. And Raspberry Pi is worked as LoRa-TCP gateway. The ESP32 is connected to Raspberry Pi and Pi transfer the gathered data to cloud server
